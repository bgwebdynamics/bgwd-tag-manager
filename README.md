# BG Web Dynamics Tag Manager

A WordPress plugin for easy Google Tag Manager integration.

## Version
Current version: 2.1.7

## Installation
1. Download the latest release
2. Upload to WordPress via Plugins → Add New → Upload Plugin
3. Activate the plugin
4. Go to Settings → BG Web Dynamics TM
5. Enter your GTM Container ID

## Automatic Updates
This plugin supports automatic updates via GitHub releases.
